# Test Plan

## Bench test before motorcycle

- lab power supply
- dummy loads
- CAN emulator/logger
- fuse test
- overcurrent test
- undervoltage test
- thermal test
- vibration inspection

## Motorcycle test

1. Install with only low-current loads.
2. Verify no CAN errors.
3. Verify ACC behavior.
4. Verify low-voltage shutdown.
5. Add fog lights.
6. Add CHIGEE/nav.
7. Log CAN without transmitting.
