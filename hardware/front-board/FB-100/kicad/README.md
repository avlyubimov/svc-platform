# FB-100 KiCad Project

Status: Preliminary schematic scaffold; no PCB layout

This directory contains a non-layout KiCad project for FB-100 schematic
planning.

There is intentionally no `FB-100.kicad_pcb` file. PCB layout, Gerbers, drills,
pick-place files, BOM/CPL order packages, and zipped manufacturing outputs
remain blocked until `../FB-100-schematic-freeze-checklist.md` is closed.
