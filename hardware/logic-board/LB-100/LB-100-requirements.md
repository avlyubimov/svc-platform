# LB-100 Logic Board Requirements

## Purpose

Replaceable computing and communication board.

## Target product MCU

STM32H563/H573 class.

## Prototype MCU

STM32F407/F429 acceptable for early firmware prototyping.

## Interfaces

- CAN1 vehicle read-only
- CAN2 expansion
- BLE
- USB-C service
- microSD
- RTC
- FRAM
- IMU
- Lux sensor
- LIN footprint
- RS485 footprint
- external inputs
